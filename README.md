# 🩺 Multi-Disease Prediction System (ML + Streamlit)

## Overview
The Multi-Disease Prediction System is a machine learning web application that predicts the risk of multiple diseases based on patient health parameters.

It provides predictions for:
- 🩸 Diabetes
- ❤️ Heart Disease
- 🧬 Breast Cancer

The project demonstrates an end-to-end ML workflow: data preprocessing → model training → saving models → deploying with Streamlit.

---

## Objective
To build an interactive ML system that:
- Accepts patient medical inputs
- Processes data using trained models
- Predicts disease risk (High / Low)
- Provides real-time results via a web interface

---

## Features
- Multi-disease prediction in a single app
- Real-time user input interface using Streamlit
- Machine learning classification models
- Clean and interactive UI
- Pre-trained models saved using Joblib
- Scalable structure for adding more diseases

---

## Machine Learning Models Used
- Random Forest Classifier
- Logistic Regression (experimentation)
- StandardScaler for feature scaling

---

## Datasets Used
- Diabetes Dataset (Pima Indians Diabetes Dataset)
- Heart Disease Dataset (UCI ML Repository)
- Breast Cancer Dataset (Wisconsin Diagnostic Dataset)

---

## Tech Stack
- Python
- NumPy, Pandas
- Scikit-learn
- Streamlit
- Joblib
